  • touchstart ou click

    • Window Phone não reconhece touchstart, Android e Firefox OS reconhece ambos os eventos.

  • Execute sempre um único evento

    • $('#botao').on('click touchstart', function(){});

    $('#botao').on('touchstart', function(){ CERTO });

  • Ajax

    • Firefox OS só executa a requisição se passarmos o argumento "mozSystem".


    var xhr = new XMLHttpRequest({mozSystem : true});


    É preciso permitir requisições XHR no manifest do Firefox Os

    "permissions" : { "systemXHR" : {} },
